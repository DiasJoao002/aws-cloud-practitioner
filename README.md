<p align="center">
    <h2 align="center">AWS Certified Cloud Practitioner</h2>
</p>

<p>Ao longo do programa de treinamento para a certificação como um todo, são abordados os conceitos relacionados a quatro principais domínios, sendo:</p>

- 01: Conceitos de nuvem
- 02: Segurança e Conformidade
- 03: Tecnologia e serviços de nuvem
- 04: Cobrança, preços e suporte

<p>Já o curso AWS Cloud Practitioner Essentials (parte da formação) se dedica exclusivamente a explicar os fundamentos da Computação em Nuvem e dos próprios serviços em nuvem da AWS. Como mencionado, o curso faz parte de um treinamento para a certificação "Cloud Practitioner - Foundational" </p>
